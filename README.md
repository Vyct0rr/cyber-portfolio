# Basic Port Scanner

## Description
This is a Python-based tool built to scan a target IP address for open ports. It helps identify accessible services on a network, which is a fundamental step in vulnerability assessment.

## Features
- Resolves hostnames to IP addresses.
- Scans a defined range of ports.
- Handles errors (like bad internet connection or user interruption).

## Technologies Used
- **Python 3**
- **Socket Library** (for networking connections)

## How to Run
1. Make sure Python is installed.
2. Run the script in your terminal:
   ```bash
   python scanner.py
